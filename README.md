# 🍽️ Feastigo End-to-End Data Analytics Project

## 📌 Project Overview

This project demonstrates the complete lifecycle of a real-world Data Analytics project using **Feastigo**, a food delivery platform, as the business case.

Instead of focusing only on dashboards, this project follows an end-to-end analytics workflow, beginning with business understanding and progressing through database design, SQL analysis, Python analytics, statistical analysis, predictive analytics, dashboard development, and business recommendations.

---

## 🎯 Project Objectives

- Understand business requirements
- Design a relational database
- Perform SQL analysis
- Analyze data using Python
- Apply statistical techniques
- Build predictive models
- Create interactive Power BI dashboards
- Generate actionable business insights
